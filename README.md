# Momentum:Build Better Habits. Maintain Momentum.


So momentum is a local Productivity Application It is offline and it runs on your machine 

It has features like reward system and task, habits, pomodoro mode.


You can also export your database

**How to Install & Run (WINDOWS ONLY)**

1.  Navigate to Releases

2.  Download the latest version

3.  And launch the app (For the first time it will create a local secure database on the same path)

4. Important: **How to Unblock:**
  
          1. Right-click `Momentum.exe` and select **Properties**.
          2. At the bottom of the **General** tab, look for the Security section.
          3. Check the **Unblock** box, click **Apply**, and run the app.

**Fallback (Run from Source):**
If Windows still refuses to let the `.exe` run, you can easily bypass the block by running the Python source code directly:
1. Open the project folder in your terminal.
2. Run `pip install customtkinter matplotlib`
3. Run `python main.py`
